для клонир,ования репозитория надо ввести команду:
git clone <ссылка на репохиторий>

для добавления файла в коммит существует команда
git add <имя файла>

для создания коммита нужна команда
git commit -m "<комментарий к комиту>"

для отправки изменений введите команду
git push

для получения изменений введите команду
git pull
